Detailed Description

The QObjectList class is a QList of QObjects.

A QObjectList is a QList<QObject>. The list can be traversed using inherited functions, e.g. getFirst(), next(), etc., or using a QObjectListIterator iterator.

See QObject::queryList() for an example of use.

See also QObjectListIterator and QObject.
